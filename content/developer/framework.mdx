# Keretrendszer

A webalkalmazást a **Next.js** keretrendszerben fejlesztettük. A **Next.js** egy *FullStack React* keretrendszer, amely lehetővé teszi a Frontend és Backend egyidejű fejlesztését egyetlen kódbázisban. Azért választottuk ezt a keretrendszert, mivel gyorsan és hatékonyan lehet vele webalkalmazásokat fejleszeni, amelyek könnyen skálázhatóak a késsőbbiekben köszönhetően az átlátható mappa alapú struktúrának.

## Példa egy Next.js alkalmazás készítésére

<Steps>

### Next.js alkalmazás létrehozása

```bash
npx create-next-app@latest
```

A fenti parancs egy úgynevezett **boilerplate**-et hoz létre, amely az alap struktúrát és fájlokat tartalmazza egy **Next.js** alkalmazásban. Ezek előtt ugyan felfog tenni pár kérdést, például a projekt nevét, a fájlok elhelyezkedését, akarunk e **TypeScript**-et vagy **Tailwind CSS**-t használni, stb.

### Next.js alkalmazás fejlesztői szerverének indítása

```bash
npm run dev
```
</Steps>
