# Fejlesztői környezet

Fejlesztői környezetnek a [**Visual Studio Code**](https://code.visualstudio.com/)-ot használtuk.

További fejlesztés során használt rendszerek:

- [Cursor](https://www.cursor.com/)
- [JetBrains WebStorm](https://www.cursor.com/)

## Hardver

Egy **Lenovo ThinkCentre M80s** PC-t használtunk a fejlesztés során amely a következőket tartalmazza:

- Processzor: **Intel core i5-10400 2.90GHz,**
- Memória: **8GB 2667 MT/s**

Operációs rendszerek, amiken dolgoztunk:

- **Windows 11 Pro x64,**
- **Arch Linux x86_64**

## Futtatási környezet:

- Node.js: **v22.13.1 (a fejlesztés idejében a legújabb LTS verzió)**
- Bun: **v1.1.45**
